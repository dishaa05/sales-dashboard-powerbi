# Sales Dashboard Power BI

## Overview
Interactive Power BI dashboard for analyzing sales performance from 2020-2022.

## Features
- KPI Cards
- Brand-wise Sales Analysis
- State-wise Sales Visualization
- Category-wise Quantity Analysis
- Profit & Cost Tracking

## Tools Used
- Power BI
- DAX
- Data Visualization
